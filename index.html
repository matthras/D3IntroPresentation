What is D3?

D3 is a Javascript library for manipulating documents based on data

  • Most robust library.
  • Steep learning curve - tutorials + exercises recommended!
    • DashingD3Tutorials.
    • Let's Make A Bar Chart.
  • Simpler alternatives: Chart.js, plot.ly.

Why should I learn D3?

  • Researchers are interested in displaying data in their papers and journals.
  • Corporate institutions often want to display interactive data on their websites.
  • Data scientists need to be able to communicate their data to their clients/stakeholders.

As web devs, chances are you'll need to interact with one or more of such professionals depending on the organisation you work for.

What advantages does D3 offer?

Let's say I want to make a bar chart.

Here's some data!
[3, 1, 4, 1, 5, 2, 9]

Instead of manually iterating through each data element:


for(i=0; i <= data.length; i++) {
	graph.appendChild(document.createElement("div")).class
}

You can just write


.data(data)
.append('div')
.attr('width', function(d) {return d;})
